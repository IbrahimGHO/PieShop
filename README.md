## Install

```sh
npm install
```

## Usage

```sh
1- Make sure that the MongoDB Compass server is running on localhost:27017 
2- run the website by typing "nodemon app.js" in the terminal
3- Open localhost:3000 in the preferred browser
```

## Run tests

```sh
npm run test
```
